# EPG

Автоматически обновляемый XMLTV EPG для каналов из IPTV-плейлиста.

После первого успешного запуска GitHub Action файл `epg.xml` появится в репозитории.

Для прямого доступа через GitHub Pages включите:
Settings → Pages → Deploy from a branch → main / (root).

После публикации адрес:
https://scorper007.github.io/epg/epg.xml

`missing_ids.txt` показывает tvg-id, для которых точного совпадения в источнике EPG не найдено.
